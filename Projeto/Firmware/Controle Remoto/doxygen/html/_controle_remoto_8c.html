ISR ( INT0_vect  )

Interrupção do botão que seleciona qual fonte de alimentação deve ser usada.

  • EIMSK &= ~(1 << INT0) - Desabilita a interrupção até enviar os dados e estes serem apresentados no lcd
  • EIFR = (1 << INTF0) - Limpa flag de solicitação de nova interrupção. evita o bounce
ISR ( ADC_vect  )

Interrupção do AD gerada pelo Overflow do Timer 0.

f_estouro = Nr_max_con * T * Prescaler Sendo:

  • Nr_max_con - 256 para o Timer 0 que é de 8 bits;
  • T - Período 1/F_CPU;
  • Prescaler - Definido nos CS02, CS01 e CS00 do registrador TCCR0B;

Para o prescaler de 1024 f_estouro = 16,384 ms

TIFR0 |= TOV0 - Limpa o flag de overflow do Timer0. Esse flag indica que houve um estouro do timer. limpar para habilitar um novo estouro para gerar a interrupção do ADC.

Interrupção da comunicação serial RX.

int main ( )

Configuração do prescaler do Timer 0. Ele está sendo usado para fazer leitura do ADC. A cada overflow gera uma interrupção.

Prescaler do Timer0
2 1 0 (CSx bits)
0 0 0 - No clock source (Timer/Counter stopped)
0 0 1 - clk I/O /(No prescaling)
0 1 0 - clk I/O /8 (From prescaler)
0 1 1 - clk I/O /64 (From prescaler)
1 0 0 - clk I/O /256 (From prescaler)
1 0 1 - clk I/O /1024 (From prescaler)
1 1 0 - External clock source on T0 pin. Clock on falling edge.
1 1 1 - External clock source on T0 pin. Clock on rising edge.
